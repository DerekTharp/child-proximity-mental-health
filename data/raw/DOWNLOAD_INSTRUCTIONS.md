# Data Download Instructions

Download the following files from https://hrs.isr.umich.edu/ (free registration required):

1. **RAND HRS Longitudinal File 2022 (V1)** -- place `randhrs1992_2022v1.dta` in this directory
2. **RAND HRS Family Data 2022 (V1)** -- place `randhrsfamr1992_2022v1.dta` and `randhrsfamk1992_2022v1.dta` in this directory
3. **HRS Cross-Wave Child Proximity** -- extract `cprox.zip` into a `cprox/` subdirectory
4. **HRS Cross-Wave Census Region/Division and Mobility File (Version 10)** -- place the extracted Stata file here as `HRSXREGION22.dta`

Alternatively, create symlinks to your existing HRS data files.
